# KyoshuBot
MK8DXのチーム挙手で役立ちそうなBot

# Description
* 1つのサーバーでチャンネルごとに募集をかけることができます
* Botがメッセージを書き換えてくれます(通知を最小限にしたつもり)

# Install
https://discord.com/api/oauth2/authorize?client_id=705559539872694272&permissions=76800&scope=bot

# Usage
* .start 明日の交流戦
* .add 21 6 交流戦
* .remove 21
* .c 21
* .d 21

# Command

.kyoshuでhelp表示
