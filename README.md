## Description
* 1つのサーバーでチャンネルごとに募集をかけることができます
* Botがメッセージを書き換えてくれます
* バグ報告・欲しい機能の提案 Issuesでお待ちしております

## Usage
hourは0-36で指定してください (時間)

needは0-30で指定してください (募集人数)

* .start 明日の交流戦
* .add 21 6 交流戦
* .remove 21
* .c 21
* .d 21
